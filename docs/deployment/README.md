# Deployment Notes

This directory holds reproducible deployment instructions for each milestone.

## M0 baseline

- No external services are required for the stub loop.
- `python3` is sufficient to run the M0 smoke path.
- Redis, Ollama, FastAPI, React build tooling, and hardware drivers remain M1+
  integrations and should be documented here when added.
